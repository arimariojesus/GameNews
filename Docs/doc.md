## Campos

* **id** - "CPF" da aplicação, identificador
* **titulo** - Nome do jogo
* **descricao** - Breve descrição do jogo
* **videoId** - Trailer do jogo

## Validação

* **id** = número > 0
* **titulo** = > 3 &&  <= 100 caracteres
* **descricao** = >= 10 && <= 250 caracteres
* **videoId** = != ""
